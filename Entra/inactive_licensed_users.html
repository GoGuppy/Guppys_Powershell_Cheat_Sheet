🔎 Inactive Licensed Users

Scan a tenant for licensed accounts with 60+ days of inactivity. Run each step in order — confirm results before taking any action.

AZ-104 Graph
Setup
1
Install Graph Module
Install once. Skip if already installed.
foundation 
Install-Module Microsoft.Graph -Scope CurrentUser
2
Connect to Microsoft Graph
Authenticate with read-only scopes for this scan.
foundation 
Connect-MgGraph -Scopes "User.Read.All", "AuditLog.Read.All"
Query
3
Pull All Licensed Active Users
Pulls all enabled, licensed accounts with sign-in data.
slow 
$Users = Get-MgUser -All `
    -Filter "accountEnabled eq true" `
    -Property "displayName, userPrincipalName, signInActivity, assignedLicenses"

$Users.Count
Note: Pulls all enabled users — licensed and inactive filtering happens in Step 4. Large tenants may take a minute.
Filter
4
Filter to Inactive Users Only
Isolates accounts past the cutoff or with no sign-in ever.
retire signal 
$InactiveUsers = $Users | Where-Object {
    $_.AssignedLicenses.Count -gt 0 -and (
        -not $_.SignInActivity -or
        [datetime]$_.SignInActivity.LastSignInDateTime -lt (Get-Date).AddDays(-60)
    )
}

$InactiveUsers.Count
Note: $null sign-in = account was created but never used. Counts as inactive.
Results & Export
5
View Results + Export to CSV
Preview in console and save to file before touching anything.
export 
$InactiveUsers | Select-Object DisplayName, UserPrincipalName,
    @{ Name = "LastSignIn"; Expression = {
        if ($_.SignInActivity) { $_.SignInActivity.LastSignInDateTime } else { "Never" }
    }} |
    Sort-Object LastSignIn |
    Format-Table -AutoSize

$InactiveUsers | Select-Object DisplayName, UserPrincipalName,
    @{ Name = "LastSignIn"; Expression = {
        if ($_.SignInActivity) { $_.SignInActivity.LastSignInDateTime } else { "Never" }
    }} |
    Export-Csv -Path ".\InactiveUsers_$(Get-Date -Format 'yyyyMMdd').csv" -NoTypeInformation

Write-Host "Done. $($InactiveUsers.Count) inactive users exported." -ForegroundColor Green
Don't be stupid!!! Review the CSV before running any disable or licensing changes against this list.
Full Script
⚡
Inactive Licensed Users — Detection Only
All steps combined. Scans and exports. Nothing gets touched.
export 
Connect-MgGraph -Scopes "User.Read.All", "AuditLog.Read.All"

$Users = Get-MgUser -All `
    -Filter "accountEnabled eq true" `
    -Property "displayName, userPrincipalName, signInActivity, assignedLicenses"

$InactiveUsers = $Users | Where-Object {
    $_.AssignedLicenses.Count -gt 0 -and (
        -not $_.SignInActivity -or
        [datetime]$_.SignInActivity.LastSignInDateTime -lt (Get-Date).AddDays(-60)
    )
}

$InactiveUsers | Select-Object DisplayName, UserPrincipalName,
    @{ Name = "LastSignIn"; Expression = {
        if ($_.SignInActivity) { $_.SignInActivity.LastSignInDateTime } else { "Never" }
    }} |
    Sort-Object LastSignIn |
    Format-Table -AutoSize

$InactiveUsers | Select-Object DisplayName, UserPrincipalName,
    @{ Name = "LastSignIn"; Expression = {
        if ($_.SignInActivity) { $_.SignInActivity.LastSignInDateTime } else { "Never" }
    }} |
    Export-Csv -Path ".\InactiveUsers_$(Get-Date -Format 'yyyyMMdd').csv" -NoTypeInformation

Write-Host "Done. $($InactiveUsers.Count) inactive users exported." -ForegroundColor Green
Don't be stupid!!! This script is detection only — review the CSV before running any disable or licensing changes.
Optional Next Steps
A
Lock Accounts from CSV
Disable every account in the exported CSV.
retire signal 
Connect-MgGraph -Scopes "User.ReadWrite.All"

Import-Csv ".\InactiveUsers_$(Get-Date -Format 'yyyyMMdd').csv" | ForEach-Object {
    Update-MgUser -UserId $_.UserPrincipalName -AccountEnabled:$false
    Write-Host "Locked: $($_.UserPrincipalName)" -ForegroundColor Red
}
Don't be stupid!!! Run detection first and confirm the CSV looks right before locking anything.
B
Randomize Passwords from CSV
Set a random password on each account in the CSV.
retire signal 
Connect-MgGraph -Scopes "User.ReadWrite.All"

Import-Csv ".\InactiveUsers_$(Get-Date -Format 'yyyyMMdd').csv" | ForEach-Object {
    $randomPw = [System.Web.Security.Membership]::GeneratePassword(20, 4)
    $pwProfile = @{ Password = $randomPw; ForceChangePasswordNextSignIn = $false }
    Update-MgUser -UserId $_.UserPrincipalName -PasswordProfile $pwProfile
    Write-Host "Password reset: $($_.UserPrincipalName)" -ForegroundColor Yellow
}
Note: Requires Add-Type -AssemblyName System.Web if the GeneratePassword call errors. Pair with locking (Step A) for full lockout.
C
Convert Accounts to Shared Mailbox
Convert each mailbox in the CSV to a shared mailbox.
retire signal 
Connect-ExchangeOnline

Import-Csv ".\InactiveUsers_$(Get-Date -Format 'yyyyMMdd').csv" | ForEach-Object {
    Set-Mailbox -Identity $_.UserPrincipalName -Type Shared
    Write-Host "Converted to shared: $($_.UserPrincipalName)" -ForegroundColor Cyan
}
Note: Requires Install-Module ExchangeOnlineManagement.
D
Remove Licenses from CSV
Strip all assigned licenses from each account in the CSV.
retire signal 
Connect-MgGraph -Scopes "User.ReadWrite.All"

Import-Csv ".\InactiveUsers_$(Get-Date -Format 'yyyyMMdd').csv" | ForEach-Object {
    $user    = Get-MgUser -UserId $_.UserPrincipalName -Property "assignedLicenses"
    $skuIds  = $user.AssignedLicenses | Select-Object -ExpandProperty SkuId
    Set-MgUserLicense -UserId $_.UserPrincipalName -AddLicenses @() -RemoveLicenses $skuIds
    Write-Host "Licenses removed: $($_.UserPrincipalName)" -ForegroundColor Yellow
}
Don't be stupid!!! Do this after converting to shared mailbox (Step C) — not before, or the mailbox conversion may fail.
