📁 Azure File Share

Az PowerShell Module · AZ-104 · Create file shares, upload files, generate SAS tokens, and map as a drive
Don't be stupid!!! Azure Files uses SMB 3.x over port 445. Many ISPs block 445 — if mapping fails from home, that's why. Quota is in GiB (max 5120 GiB for standard). SAS tokens grant access without a key — treat them like passwords. Always delete test shares and resource groups when done.
