⚖️ Azure Load Balancer

Az PowerShell Module · AZ-700 · Deploy a public load balancer with a backend pool and load balancing rule
Don't be stupid!!! Make sure to test first before using in production. Standard SKU LB requires Standard SKU public IPs — mixing Basic and Standard SKUs will error out. If you see module errors, run Update-Module Az and restart PowerShell.
