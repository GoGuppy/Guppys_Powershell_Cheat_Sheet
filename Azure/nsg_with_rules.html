🛡️ Azure NSG with Rules

Az PowerShell Module · AZ-700 · Create an NSG, define rules, and associate to a NIC or subnet
Don't be stupid!!! Make sure to test first before using in production. Always clean up inbound rules (RDP/SSH) when not in use. Priority lower number = higher priority — plan your rule order before deploying. If you see module errors, run Update-Module Az and restart PowerShell.
