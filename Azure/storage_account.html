🗄️ Storage Accounts

Az PowerShell Module · AZ-104 · Blob storage, containers, keys, access tiers, and lifecycle
Storage account names must be globally unique, 3–24 characters, lowercase letters and numbers only. SKU controls replication: Standard_LRS (cheapest, local), Standard_GRS (geo-redundant), Standard_ZRS (zone), Premium_LRS (SSDs). Kind StorageV2 is the current default — use it unless you have a specific reason not to.
