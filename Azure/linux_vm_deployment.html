🐧 Azure Linux VM Deployment

Az PowerShell Module · AZ-104 · Ubuntu VM build, NSG, SSH, and teardown · run PowerShell as Administrator
Don't be stupid!!! Make sure to test first before using in production. Always delete the NSG SSH rule (step 14) when not actively using SSH. Remove the resource group (step 17) when done to stop billing. If you see module errors, run Update-Module Az and restart PowerShell.
